## Problema

Los logs no aparecen en New Relic para un CloudWatch a través de la configuración de integración de Firehose.

## Solución

Posibles razones y sus soluciones:

* La clave de API puede no ser correcta. Navegue a <DNT>**Amazon Data Firehose &gt; Firehose streams &gt; your-stream &gt; Edit destination settings**</DNT> y actualice la clave de API con la clave de licencia de ingesta correcta.
* El extremo proporcionado puede ser incorrecto. Verificarlo.
* Consulte los logs de S3 o los logs de errores de Cloudwatch.
