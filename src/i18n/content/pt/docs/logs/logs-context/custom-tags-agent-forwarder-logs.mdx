O agente de linguagem New Relic APM agora permite que você opte por adicionar sua tag personalizada (rótulos) ao log encaminhado pelo agente. Com a tag personalizada no log, DevOps e o engenheiro de plataforma podem filtrar, pesquisar e correlacionar dados log para uma resolução de problemas mais rápida e eficiente, melhor desempenho e utilização otimizada de recursos.

## Habilitar tag personalizada (rótulos) no log do aplicativo

Quando habilitado, o agente de idiomas da New Relic adiciona todas as tags personalizadas (rótulos) ao log encaminhado pelo agente.
