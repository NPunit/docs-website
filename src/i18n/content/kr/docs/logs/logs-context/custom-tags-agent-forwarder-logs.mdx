뉴렐릭 APM 언어 에이전트를 사용하면 에이전트 전달 로그에 사용자 정의 태그(레이블)를 추가하도록 선택할 수 있습니다. 로그에 사용자 정의 태그를 사용하면 DevOps 및 플랫폼 엔지니어가 로그 데이터를 필터링, 검색 및 연관시켜 더 빠르고 효율적인 문제 처리, 향상된 성능 및 최적화된 리소스 활용을 수행할 수 있습니다.

## 애플리케이션 로그에서 사용자 정의 태그(레이블) 활성화

활성화되면 뉴렐릭 언어 에이전트가 모든 사용자 정의 태그(레이블)를 에이전트 전달 로그에 추가합니다.

지원 언어:

* [.그물](/docs/apm/agents/net-agent/configuration/net-agent-configuration)
* [루비](/docs.newrelic.com/docs/apm/agents/ruby-agent/configuration/ruby-agent-configuration/)
* 이동: 곧 출시
* 자바: 곧 출시
* Node.js: 곧 출시
* PHP: 곧 출시
* 파이썬: 곧 출시
