### New docs

* Gain security insights with FOSSA in New Relic Vulnerability Management dashboard. [Quickly view FOSSA open-source vulnerability scan results alongside other security detections within the New Relic Vulnerability Management dashboard.](https://docs.newrelic.com/whats-new/2024/12/whats-new-12-03-fossa/) 
* Introduced two new features namely 'last viewed entities' in Error Inbox and 'improved login flow' error responses. [Refer the section for mobile app support and download link for Android v5.26.1](https://docs.newrelic.com/docs/release-notes/mobile-apps-release-notes/new-relic-android-release-notes/new-relic-android-5261/)

### Major change

* Created docs that explain the new auto collected logs in the iOS agent. [Added information on the newly created feature of the iOS agent to collect the stdout and stderr output of the apps it instruments.](https://docs.newrelic.com/docs/mobile-monitoring/new-relic-mobile-ios/configuration/ios-remote-logging/)
* Updated New Relic Service Broker Support for TAS (Tanzu Application Service) 6.0 [Tabular data updated.](https://docs.newrelic.com/docs/infrastructure/other-infrastructure-integrations/cloudfoundry-integrations/vmware-tanzu-service-broker-integration/)

### Minor changes

* Updated the support date for Azure SQL database. [Azure is deprecating the SDK the polling API uses and will no longer be providing critical updates to this API, creating security concerns moving forward.](https://docs.newrelic.com/eol/2024/04/eol-04-24-24-polling-api/)
